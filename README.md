# Overview

Snap Grab Swift backend. Current project is based on NestJS framework.
Please read NestJS documentation for more information.

###### Tech Stack ######

**Framework:**
*  NestJS

**Databases:**
*  Mysql (TypeORM)
*  Elasticsearch

**Payment Gateways:**
*  Stripe (for Stripe and Apple Pay)
*  PayPal

**Push Notifications:**
*  OneSignal

**SMS Service:**
*  Twillio

**Email Sender Service:**
*  SendGrid

**Geocoder Service:**
*  Google Maps

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
